# Elite Dangerous Photographer

Elite Dangerous Photographer — это продвинутый инструмент для автоматизации жизни исследователя космоса. Хватит гадать, где был сделан тот или иной красивый скриншот! EDP подхватывает новые снимки «на лету», вытягивает данные из игровых журналов и превращает стандартные Screenshot_0001.png в информативные файлы с именами систем, тел и даже координат.

## Ключевые функции

* Мониторинг в реальном времени: Приложение отслеживает выбранную папку и автоматически переименовывает новые скриншоты
* Автоматическое извлечение данных:
  * Текущая звездная система;
  * Небесное тело (планета/звезда);
  * Точные координаты поверхности или название станции при стыковке.
* Поддержка автоматического преобразования тяжелых .bmp в компактные .png или .jpg
* Возможность автоматически раскладывать скриншоты по папкам с названиями систем и выбор отдельной папки для переименованных изображений
* Встроенный просмотрщик с сеткой и списком, быстрым предпросмотром при наведении
* Интеграция с буфером обмена. Копируйте сам скриншот или только координаты места одной кнопкой

## Настройка (Формат имени)

Вы можете полностью настроить порядок элементов в названии файла:

1. Дата (ГГГГ-ММ-ДД)
2. Время (ЧЧ-ММ-СС)
3. Небесное тело
4. Координаты
5. Имя КМДР

### Пример итога: 2026-04-25 15-19-10 (Phrua Flyuae ZE-A e7 — A 3 a — [46.29, -120.97]) Explorer123

Ваша история заслуживает красивых имен. Fly safe, Commander! o7
